# Sutradhar Planning Index

This directory is the local planning source for building `sutradhar` as a centralized protobuf contract platform.

## Primary Files

- `sutradhar-grill-decision-log.md`
  - Full decision history and accepted architecture matrix.
- `sutradhar-comprehensive-implementation-plan.md`
  - Execution-ready implementation roadmap with milestones, definitions of done, acceptance criteria, CI gates, release model, and intern operating instructions.

## Reading Order For New Implementers

1. `sutradhar-grill-decision-log.md`
2. `sutradhar-comprehensive-implementation-plan.md`

## Execution Rule

Implement from the comprehensive plan unless a newer decision is appended in the decision log. If there is a conflict, decision log wins and the comprehensive plan must be updated in the same task.
